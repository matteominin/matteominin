---

I build reliable backend systems and multi-agent LLM pipelines in **Java / Spring**. I shipped
[**horsereport.ai**](https://horsereport.ai), an AI platform that cut expert horse-evaluation
turnaround from a full day to a **~70-second median**, and co-authored
[**CAPRA** (CSEE&T 2026)](https://arxiv.org/abs/2606.18976) on scaling architecture feedback with a
multi-agent LLM system.

- 🔭 **Now:** demand paging in the os161 kernel (C) · reliability engineering for production LLM systems
- 🌱 **Open source:** fixed a silent message-drop bug in Apache Camel's Kafka producer ([merged, PR #24467](https://github.com/apache/camel/pull/24467), ships in 4.22.0)
